Principais formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b> (não semântica).

Nesta frase temos um termo em destaque usando a tag <strong> (semântica).

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag <i> (não semântica).

Nesta frase temos um termo em ênfase usando a tag <em> (semântica).

Texto marcado

Podemos criar também um texto marcado usando a tag <mark>.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo.
A tag <big> está absoleta. Já a tag <small> ainda está em uso.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado, utilizando a tag <del>

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, utilizando a tag <ins>.
Existe também o subinhado com a tag <u> (não semântica).

Texto sobrescrito

Para inserir coisas como a potenciação. Exemplo: x20+3 , utilizando a tag <sup>.

Texto subscrito

para inserir coisas como esta: H2O , utilizando a tag <sub>.
